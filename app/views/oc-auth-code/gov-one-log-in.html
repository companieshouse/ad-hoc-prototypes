Create your GOV.UK One Login or sign in

You can use your GOV.UK One Login to access some government services. In the future, you’ll be able to use it to access all services on GOV.UK.

You'll need:

  • an email address
  • a way to get security codes - this can be a mobile phone number or an authenticator app

You can also use GOV.UK One Login in Welsh (Cymraeg).
